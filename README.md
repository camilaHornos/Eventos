🤖 Automatización para optimizar transferencias de stock entre ubicaciones

Desarrollé una herramienta en Python para resolver situaciones donde determinados productos tienen alta demanda en fechas específicas y no siempre se cuenta con stock suficiente en el depósito principal.

El análisis permite:
• Detectar productos con alta demanda en días específicos
• Identificar ubicaciones con stock insuficiente
• Buscar stock disponible en otras ubicaciones cercanas
• Calcular transferencias sin dejar sin stock a las ubicaciones origen
• Priorizar según nivel de ventas
• Generar automáticamente un reporte con sugerencias de transferencia

Este tipo de automatización mejora la disponibilidad de productos durante eventos o picos de demanda, optimizando la distribución del inventario sin afectar el abastecimiento general.

Tecnologías utilizadas: Python, Pandas, análisis de datos y automatización.
